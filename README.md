# IRPay, Manage Your Pays Easily!

With IRPay you can simply implant pay getaways. This library works standalone and with frameworks like express, etc.

## Installation

Install IRPay with npm

```bash
  npm install irpay
```

Or Yarn:

```bash
  npm install irpay
```

PNPM:

```bash
  pnpm add irpay
```
